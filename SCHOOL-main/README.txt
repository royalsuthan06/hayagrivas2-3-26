Hayagrivas International School Website Project
=============================================

This project follows a professional Model-View-Controller (MVC) inspired structure separating Frontend, Backend, and Assets.

DIRECTORY STRUCTURE
-------------------
/hayagrivas-project
│
├── frontend/          (User Interface - HTML Pages)
│   ├── index.html     (Home Page with Video Hero)
│   ├── about.html
│   ├── gallery.html
│   ├── admissions.html
│   ├── contact.html
│   └── ...
│
├── backend/           (Server Logic - PHP)
│   ├── config/        (Database Configurations)
│   ├── api/           (API Endpoints)
│   └── utils/         (Helper Functions)
│
└── assets/            (Static Resources)
    ├── css/           (Stylesheets)
    ├── js/            (JavaScript Logic)
    └── media/         (Images & Videos)

FEATURES
--------
1. Premium Animations (Opening Loader, Ripple, Hover Effects).
2. Responsive Layout (Mobile Friendly).
3. Backend Connection Ready (Forms connected to PHP APIs).
4. Real-time Map and Contact Details.

USAGE
-----
1. Open `frontend/index.html` in a browser.
2. For Backend features, host the project on a local server (XAMPP/WAMP).
